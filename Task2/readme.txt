Тут только созданная и заполненная база, на реализацию не хватило времени

План выполнения:
По заданию 2:
1) создаем клиент-серверное приложение, клиент используется для управления, сервер работает с БД
2) функции добавления, удаления, редактирования и поиска реализуются на сервере
3) клиент только отправляет название команд и параметры запроса 


Реализация функции добавления:
1) подпрограмма поиска в БД уже существующей записи с уникальными введенными параметрами (номер телефона, номер лицензии)
2) добавление записи, если не найдено совпадений, сообщение об ошибке, если были найдены совпадения
3) отправка сообщения "запись добавлена" (если запись была добавлена)

Реализация удаления:
1) поиск в БД
2)удаление, если запись найдена
3)если запись не найдена сообщить об этом

Поиск по фио (др. параметров):
1) передача параметра на сервер
2) формирование запроса на поиск (на стороне сервера), с использованием полученных параметров
3) возвращение клиенту найденных записей или сообщения об их отсутствии в БД
